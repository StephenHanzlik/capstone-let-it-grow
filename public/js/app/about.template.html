Let It Grow!

Browser based garden managment


Let It Grow is an open sourced browser based project that allows users to pull data from indoor and outdoor gardens. It was inspired by the concept of the vertical farm, food production it vertically aligned rows, such as a sky scraper, and is intedended to make accessing facility data easier. An Arduino Uno, combined with a powerful weather sheild from Spark Fun and WIFI capabilities by Electric Imp, sends HTTP post requests to the server every second. Charts are used to vizualize temperature, humidity, light, and soil moisture data in real time. This project exemplifies the power of the modern browser and it's ability to render a single page app powered by JSON data. Let it grow, let it grow, greatly yield!

Get Data!

Let It Grow utilizes an Arduino Microcontroller and a wifi shield to post data to the server. Live graphs and gauges give users insight into current grow conditions and foster decisive, informed decisions. users will be able to fine tune the inputs conducive to a successful harvest.
